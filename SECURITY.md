# Security Policy

## Supported Versions

We actively update and patch security vulnerabilities for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please report it immediately:

1. **Do not open a public issue.**
2. Send an email report to the project owner at `[security email placeholder]`.
3. Include a description of the vulnerability, steps to reproduce, and any proof-of-concept code or screenshots.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue before making any public announcement.
