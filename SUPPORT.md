# Support Policy

Thank you for using the Electro Store! Here is a guide on where to get support, find answers, and ask questions.

## How to Get Support

### 1. Check the Documentation
Before opening a ticket or support thread, please review:
- The [README](README.md) for quick-start configuration, tech stack details, and features.
- In-line comments in typescript components explaining module routing and imports.

### 2. GitHub Issues
If you encounter errors, crashes, or bugs:
- Search the open and closed issues on GitHub to see if someone else has already reported it.
- If it hasn't been reported, open a new Issue using the templates provided.

### 3. Discussions
For general questions, configuration help, and suggestions:
- Use the **Discussions** tab on the repository to post your question or share your visualizer customizations.

---

## Response Times

This is a free, open-source project. While we strive to review issues and pull requests quickly, response times are not guaranteed. For critical bugs or pull requests, please expect a response within a few business days.
